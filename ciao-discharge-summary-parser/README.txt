CIAO Discharge Summary Parser

To run the parser application:

1) First check that Java is installed on your machine. Java SE 6 or higher is required - see http://www.java.com
2) Double click on ${project.artifactId}-${project.version}.jar to launch the GUI application.

Alternatively the application can be started from the command line via:
	java -jar ${project.artifactId}-${project.version}.jar <inputFolder> <outputFolder>
